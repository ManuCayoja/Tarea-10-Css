Layout - display: inline-block

La propiedad display: inline-block en CSS se utiliza para hacer que un elemento se comporte como un elemento en línea, pero manteniendo las características de un elemento en bloque. Esto significa que el elemento se coloca en línea con los elementos adyacentes, pero aún puede tener dimensiones (ancho, alto, margen, relleno, borde) y ser manipulado como un elemento de bloque.
